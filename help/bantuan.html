FAQ

1. Atas tujuan apa website ini dibuat?

Project kecil yang bermaksud sebagai latihan saya dalam pengaplikasian materi DOM dan JavaScript lainnya.

2. Dengan bahasa pemrograman apa website ini dibuat?

Website ini dibuat menggunakan HTML, CSS, dan JavaScript vanilla.

3. Bagaimana cara mengkonversi unit?

Masukkan angka, pilih satuan asal dan satuan tujuan, lalu klik tombol "Convert".

4. Mengapa ada beberapa konversi yang unitnya sedikit?

karena sulit untuk mengimplementasikan unit lainya, namun bisa saja diupdate berkala nantinya.

5. Apakah ada konversi unit baru kedepannya?

Ya, jika website ini ramai dikunjungi, ataupun saya yang sedang gabut.
